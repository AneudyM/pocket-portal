# Overview

The [Pocket Portal](https://www.portal.pokt.network/) is a server side rendered web application built on [Typescript](https://www.notion.so/Typescript-4b82f284f60e483982a55156cd4c8271) with [remix.run](http://remix.run) being the framework. We pull a lot of common components from our [component library](https://www.notion.so/Pocket-Blocks-a08dde7b4bb54ed7b497a03a63d3fd59). We also implement a lot of testing in various areas to make every effort to keep our code consistent and to avoid bugs / breaking changes.
